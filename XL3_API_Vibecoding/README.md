# XL3 Basic Streaming API – Webinar Prompts

A collection of prompts from today's webinar demonstrating how to use AI to build a WebSocket dashboard for the XL3 Basic Streaming API.

---

## Prompts

### 🇬🇧 English

> Create an HTML/JS dashboard that connects to the XL3 Basic Streaming API via WebSocket. The dashboard should:
> - Display all 123 measurements and the spectrum data in real time
> - Derive its entire UI structure dynamically from the **settings message**
> - Follow the attached Markdown specification exactly
> - Allow the user to enter the **connect key** and **password** before establishing the connection

---

### 🇩🇪 Deutsch

> Erstelle ein HTML/JS-Dashboard, das über WebSocket eine Verbindung zur XL3 Basic Streaming API herstellt. Das Dashboard soll:
> - Alle 123 Messwerte sowie die Spektrumdaten in Echtzeit anzeigen
> - Die gesamte UI-Struktur dynamisch aus der **Settings-Nachricht** ableiten
> - Die beigefügte Markdown-Spezifikation exakt einhalten
> - Dem Benutzer ermöglichen, **Verbindungsschlüssel** und **Passwort** vor dem Verbindungsaufbau einzugeben

---

## Notes

- Both prompts describe the same task – use whichever language fits your workflow.
